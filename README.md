# doesntpad
Doesntpad is an online notepad made in Rust inspired by `http://dontpad.com/`.

## Requirements
* Rust Nightly

## Quickstart
1. Clone this repository to your local.
> `https://github.com/hermanodecastro/doesntpad.git`

2. Run `cargo run` to start the application.
